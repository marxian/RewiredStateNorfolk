Rewired State runs hack days to let developers show government what is possible and government show developers what is needed. Neontribe is a Norfolk web development company. We were one of four hosting organisations for Young Rewired State 2010. We're delighted to be hosting the first Rewired State event that's “arms-length but authorised”.

At 6.30, on May 7th, in the beautiful Octagon Chapel (pictured), the developers who've signed up will show what we've been doing with Norfolk data.

We're expecting you to be surprised and delighted by what's possible in a day. And we hope you'll help some of the projects go further. If you'd like to come along, please let us know here.
